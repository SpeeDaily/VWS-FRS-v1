# Dynamic Content Loader and Recommendation System

A dynamic content loader and recommendation system for frontend websites.

## Features
- Dynamically loads content recommendations based on user clicks.
- Recommends content based on user interests.
- Shuffles and displays content items.

## Demo
Check out the live demo [here](https://speedaily.github.io/VWS-FRS-v1/).
